# Ímpar ou Par

Jogo web **Ímpar ou Par** desenvolvido com **HTML**, **CSS** e **JavaScript** como exercício de lógica de programação.

## Sobre o projeto

O jogador escolhe entre **Ímpar** ou **Par**, digita um número de 1 a 10 e o sistema gera um número aleatório para o adversário **João**. A soma dos dois números define o vencedor da rodada.

## Como funciona

1. Selecione **Ímpar** ou **Par**
2. Digite um número entre **1** e **10**
3. Clique em **Jogar**
4. O sistema sorteia o número do João e calcula a soma
5. Se o resultado (par ou ímpar) coincidir com sua escolha, **você ganha**

## Funcionalidades

- Escolha entre ímpar ou par com radio buttons
- Validação de número (1 a 10)
- Número aleatório para o adversário
- Histórico de partidas na tela
- Excluir resultados individualmente
- Persistência do histórico com **localStorage**

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage (Web API)
